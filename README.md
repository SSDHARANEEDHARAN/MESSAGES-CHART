# Character Counter Application

This is a simple web application that allows users to input a message and track the number of characters they have typed. The application includes a responsive design and character limit functionality.

## Features

- **Character Count**: Users can type up to 200 characters.
- **Responsive Design**: The application adapts to different screen sizes, ensuring usability on mobile and desktop devices.
- **User-friendly Interface**: Clear labels and a counter display make it easy for users to see their input.

## Technologies Used

- HTML
- CSS
- JavaScript
